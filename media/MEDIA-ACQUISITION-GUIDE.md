## Phase 1: NARA (National Archives) - FREE PUBLIC DOMAIN

**Priority:** HIGHEST - Start here for all historical photos

### Step 1: Access NARA Digital Archive

1. **Go to:** https://catalog.archives.gov/
2. **Click:** "Search the Catalog"
3. **Set Filters:**
   - Digital Images: YES
   - Date Range: 1940-1945
   - Record Group: 111 (Signal Corps) or 407 (Adjutant General)

### Step 2: Philippine Scouts Photo Search

**Search Query Sequence** (try each one):

```
Search 1: "Philippine Scouts" 1941
Search 2: "Philippine Division" insignia
Search 3: "26th Cavalry Regiment" Philippines
Search 4: "24th Field Artillery" Philippines
Search 5: Philippine AND Scouts AND training
Search 6: Philippine AND Division AND 1941
Search 7: "57th Infantry" Philippines
Search 8: "45th Infantry" Philippines
Search 9: Bataan AND defenders
Search 10: Philippine AND Commonwealth AND Army
```

### Step 3: Download Process

For each relevant photo:
1. **Click image** to open full record
2. **Check "Use Restrictions"** - should say "Unrestricted" for public domain
3. **Download highest resolution** available
4. **Record in spreadsheet:**
   - NARA ID number
   - Description
   - Date (if known)
   - Photographer (if credited)
   - Which video script it's for

### Step 4: Specific Images to Find

**High Priority - Must Have:**
- [ ] Philippine Scouts in formation showing carabao patch (Script #1 hero shot)
- [ ] Close-up of carabao shoulder patch on uniform (Script #1, #2)
- [ ] American and Filipino officers together (Script #1)
- [ ] 26th Cavalry Regiment - mounted or dismounted (Script #3)
- [ ] Artillery position/crew (24th FA preferred) (Script #3)
- [ ] Defensive positions on Bataan (Script #3)

**Medium Priority - Nice to Have:**
- [ ] Training exercises pre-war
- [ ] Equipment/weapons close-ups
- [ ] Camp scenes
- [ ] Medal of Honor recipients (search individual names)

### Step 5: Medal of Honor Recipients Photos

**Individual Searches:**
```
Jose Calugas Medal Honor
Alexander Nininger Medal Honor Philippines
Willibald Bianchi Medal Honor
```

**Alternative Sources:**
- Congressional Medal of Honor Society website
- Individual NARA personnel file searches
- Military awards databases

### Expected Results
- **Time needed:** 2-4 hours of searching
- **Photos expected:** 20-50 relevant images
- **Success rate:** ~60% of needs met from NARA alone

## Phase 2: Military Insignia - FREE PUBLIC DOMAIN

### Institute of Heraldry

**Website:** https://tioh.army.mil/

**Steps:**
1. **Search insignia database** for "Philippine Division"
2. **Download official insignia artwork** if available
3. **Note:** May need to submit request form for high-res files

**Alternative: Contact directly**
- Email: tioh.info@mail.mil (check current contact)
- Request: Official Philippine Division shoulder sleeve insignia artwork

### Wikipedia Commons

**Search URL:** https://commons.wikimedia.org/

**Search Terms:**
```
"Philippine Division" insignia
Philippine Scouts patch
US Army Philippines WWII insignia
Carabao insignia
```

**Steps:**
1. Search each term
2. Check **license** - must be public domain or CC-BY
3. Download **highest resolution**
4. **Save attribution info** from page

**What you need:**

**Where to commission:**
- Fiverr: Search "vector insignia recreation" ($20-50)
- Upwork: Military graphics specialists ($50-150)
- 99designs: Design contest ($200-400)

2. Search: "Philippines" AND "map" AND 1941
3. Filter: "Cartographic Materials"
4. Look for: "USAFFE" (United States Army Forces Far East)

1. Go to: https://www.davidrumsey.com/

### Create Custom Tactical Maps

**Option 1: Use QGIS (Free Software)**

**Download:** https://qgis.org/

**Steps:**
1. Install QGIS
2. Add base layer: OpenStreetMap or satellite
3. Add historical map overlay (from sources above)
4. Draw defensive lines, unit positions
5. Export as high-res PNG

**Tutorials:**
- YouTube: "QGIS military map" tutorials
- QGIS documentation: Map making basics

**Option 2: Commission Custom Maps**

**Where:**
- Fiverr: Search "military map" or "tactical diagram"
- Upwork: Cartographers with military experience
- Military graphics specialists

**Cost:** $100-300 per detailed tactical map

**Provide them:**
- Reference historical maps
- Unit positions (from historical research)
- Specific battle phases to show
- Color scheme (red/blue standard)

## Phase 5: Tactical Diagrams & Animations - CREATE/COMMISSION

### DIY Approach: PowerPoint/Keynote

**Surprising but effective for simple diagrams:**

1. **Use PowerPoint/Keynote**
2. **Insert shapes:** Rectangles for units, arrows for movements
3. **Color code:** Blue (US/Filipino), Red (Japanese)
4. **Labels:** Clear unit designations
5. **Export:** High-res PNG or PDF

**Pros:** Free, easy, works for simple diagrams
**Cons:** Not professional-looking, no complex animation

### Professional Tools: After Effects

**If you have Adobe subscription:**

1. **After Effects** for animations
2. **Illustrator** for static diagrams
3. **Tutorials:** YouTube "military tactical animation After Effects"

**Templates:**
- VideoHive: Search "military map animation"
- Motion Array: Tactical/military templates

### Commission Professional Tactical Animator

**When to commission:**
- Complex animations (fields of fire, counterattacks)
- Professional polish needed
- Limited time/skills for DIY

**Where to find:**
- **Fiverr:** "military animation" "tactical diagram"
- **Upwork:** Motion graphics specialists with military experience
- **Military gaming communities:** May know skilled animators

**Cost:** $300-1000 per minute of animated content

**What to provide:**
1. Historical maps and references
2. Detailed script of what to show when
3. Color preferences
4. Example videos you like

**Why:** Best for YouTube/social media, affordable subscription

**Steps:**
1. **Go to:** https://www.epidemicsound.com/
2. **Sign up:** 30-day free trial available
3. **Search tracks:**
   - Script #1: "military" "march" "patriotic"
   - Script #2: "ethnic" "Asian" "cultural"
   - Script #3: "tension" "war" "tactical"
4. **Download:** Unlimited with subscription
5. **Cost:** ~$15/month (personal), ~$25/month (commercial)

### Alternative: Artlist

**Similar to Epidemic Sound**
- **URL:** https://artlist.io/
- **Cost:** ~$15/month
- **License:** Lifetime license for downloaded tracks
- **Good for:** Cinematic, high-quality tracks

### Budget Option: YouTube Audio Library

**Free but limited:**
1. Go to: YouTube Studio > Audio Library
2. Search by mood/genre
3. Download: Free
4. **Limitations:**
   - Smaller selection
   - Less professional quality
   - Many tracks overused

### Filipino-Inspired Music

**Challenge:** Hard to find authentic Filipino instrumentation in stock libraries

**Options:**
1. **Commission Filipino musician** (Fiverr, SoundBetter)
   - Cost: $200-500 for 60-90 second track
   - Get: Authentic kulintang, bamboo instruments
2. **Philippine music archives**
   - Contact cultural organizations
   - May have public domain recordings
3. **Creative Commons search**
   - Search: "Filipino music" "kulintang" with CC license filter

## Phase 7: Text Overlays & Graphics - CREATE

### Tools (Choose One)

**Canva (Easiest):**
- **URL:** https://www.canva.com/
- **Free version:** Sufficient for text overlays
- **Templates:** Search "military" or start blank
- **Export:** PNG with transparent background

**Adobe Illustrator (Professional):**
- Vector graphics
- Perfect text control
- Requires subscription and learning curve

**GIMP (Free Photoshop Alternative):**
- **URL:** https://www.gimp.org/
- Free and powerful
- Steeper learning curve than Canva

### Design Specifications

**Color Scheme:**
- Primary: Red (#C41E3A) and Gold (#FFD700) - from carabao insignia
- Text: White or cream for readability
- Background: Semi-transparent black (for overlay on footage)

**Fonts to Use:**
- **Titles:** Stencil fonts (Impact, Bebas Neue, Oswald Bold)
- **Body text:** Clean sans-serif (Arial, Helvetica, Roboto)
- **Avoid:** Overly decorative or hard-to-read fonts

**Text Overlay Checklist:**

**Script #1:**
- [ ] "Formed: 1901"
- [ ] "Strength in 1941: 11,972 soldiers"
- [ ] "Elite US Army units"
- [ ] "Filipino soldiers, American command"
- [ ] "Follow for more stories of Filipino heroes in WWII"

**Script #2:**
- [ ] Organizational chart (see structure in requirements doc)
- [ ] "Three Philippine Scouts won the Medal of Honor at Bataan"
- [ ] End card text

**Script #3:**
- [ ] "Defensive Line Tactics"
- [ ] "24th Field Artillery Regiment (PS)"
- [ ] "26th Cavalry Regiment (PS)"
- [ ] "Battle of the Points: Jan 22-Feb 13, 1942"
- [ ] Medal of Honor recipients (names and units)
- [ ] "Follow for more tactical history"

### Creation Process

1. **Open Canva** (or your tool)
2. **Set dimensions:** 1080x1920 (vertical video) or 1920x1080 (if horizontal)
3. **Create each text overlay** on separate file/artboard
4. **Export:** PNG with transparent background
5. **Name clearly:** script1_formed_1901.png
6. **Organize:** Save to `media/graphics/text-overlays/`

## Common Pitfalls to Avoid

### Copyright Issues
- ‚ùå **Don't** assume old photos are public domain - check each one
- ‚ùå **Don't** use stock photos without license for commercial use
- ‚ùå **Don't** use modern Philippine military insignia (may be trademarked)
- ‚úÖ **Do** document every source and license
- ‚úÖ **Do** keep attribution info even for public domain

### Quality Issues
- ‚ùå **Don't** use low-resolution images that will pixelate
- ‚ùå **Don't** over-compress video footage
- ‚ùå **Don't** use watermarked stock without license
- ‚úÖ **Do** get highest resolution available
- ‚úÖ **Do** test graphics at final video size

### Historical Accuracy
- ‚ùå **Don't** use photos from wrong time period (post-1945)
- ‚ùå **Don't** use photos from wrong theater (European theater, etc.)
- ‚ùå **Don't** mislabel units or locations
- ‚úÖ **Do** verify photo captions against historical records
- ‚úÖ **Do** cross-reference dates and locations

### Licensing Mistakes
- ‚ùå **Don't** use "Editorial Use Only" footage for documentary
- ‚ùå **Don't** exceed usage terms (e.g., using stock in multiple projects without proper license)
- ‚ùå **Don't** forget to download license certificates
