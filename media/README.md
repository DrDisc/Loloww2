# Media Assets Directory

This directory contains all media assets for the Lolo's War video series.

## Directory Structure

```
media/
├── photos/                    # Historical photographs
│   ├── philippine-scouts/     # Philippine Scouts in uniform, training, etc.
│   ├── insignia/              # Close-ups of patches and insignia
│   ├── bataan/                # Bataan campaign, defensive positions
│   └── medal-of-honor/        # Medal of Honor recipients
├── footage/                   # Video footage
│   ├── historical/            # WWII-era footage from archives
│   └── b-roll/                # Modern footage (carabao, landscapes, etc.)
├── maps/                      # Cartographic materials
│   ├── static/                # Static map images
│   └── animated/              # Animated tactical maps
├── graphics/                  # Created graphics and overlays
│   ├── diagrams/              # Tactical diagrams, org charts
│   ├── text-overlays/         # Text graphics for video overlay
│   └── insignia/              # Vector/high-res insignia recreations
├── music/                     # Licensed music tracks
└── sound-effects/             # Sound effects (artillery, horses, etc.)
```

## File Naming Conventions

### Photos
- Format: `[subject]_[detail]_[number].jpg`
- Example: `scouts_uniform_carabao_patch_01.jpg`
- Example: `26th_cavalry_mounted_01.jpg`

### Footage
- Format: `[subject]_[duration]s_[source].mp4`
- Example: `carabao_rice_field_15s_pexels.mp4`
- Example: `bataan_defense_archive_30s_nara.mp4`

### Maps
- Format: `[location]_[type]_[date].png`
- Example: `bataan_defensive_lines_1942.png`
- Example: `philippines_overview_modern.png`

### Graphics
- Format: `[script]_[type]_[description].png`
- Example: `script1_text_formed_1901.png`
- Example: `script3_diagram_fields_of_fire.png`

### Music
- Format: `[script]_[mood]_[duration]s.mp3`
- Example: `script1_military_cadence_90s.mp3`
- Example: `script2_filipino_inspired_75s.mp3`
