# Media Directory for Lolo's War Project

This directory contains visual media supporting the historical research on the Philippine Scouts, Bataan Death March, and WWII Pacific Theater.

## Directory Structure

- `philippine-scouts/` - Images of Philippine Scouts, uniforms, and training
- `bataan/` - Images related to the Bataan peninsula defense and Death March
- `insignia/` - Military unit insignia, especially the Carabao Division
- `maps/` - Maps of the Pacific Theater, Philippines, and Bataan
- `veterans/` - WWII veteran photographs
- `historical/` - Pre-war Philippines and colonial period images

## Sources

All images are sourced from public domain or appropriately licensed materials:

- Wikimedia Commons
- National Archives and Records Administration (NARA)
- Public domain historical photographs
- Wikipedia articles

## Usage Notes

- Images are used for educational and historical documentation purposes
- All images include attribution when available
- Images are optimized for web use while maintaining historical accuracy
