## üó∫Ô∏è NARA Catalog Search (1-2 hours)

### ASSET 7: Philippine Scouts Photos from NARA
**What:** Original Signal Corps photos, RG 111
**Time:** 1-2 hours
**URL:** https://catalog.archives.gov/

**Search Steps:**
1. Go to: https://catalog.archives.gov/
2. Click "Advanced Search"
3. Enter parameters:
   - **Keywords:** "Philippine Scouts"
   - **Begin Date:** 1941-01-01
   - **End Date:** 1942-12-31
   - **Record Group:** 111
   - **Digital Copies:** Check "Available Online Only"
4. Click "Search"

**For each result:**
1. Click thumbnail to view
2. If relevant, click "Download" link
3. Save highest resolution available
4. Record NARA ID number (e.g., 111-SC-12345)
5. Save to appropriate folder

**Additional searches to run:**
```
Search 2: "26th Cavalry Regiment" (Date: 1940-1942, RG 111)
Search 3: "24th Field Artillery" AND "Philippines" (Date: 1940-1942, RG 111)
Search 4: "Bataan" (Date: 1942-01-01 to 1942-04-30, RG 111)
Search 5: "57th Infantry" AND "Philippine" (Date: 1940-1942)
Search 6: "45th Infantry" AND "Philippine" (Date: 1940-1942)
```

**Target:** Find 15-25 good quality photos

## üÜò TROUBLESHOOTING

### Can't download from PICRYL
- Try different browser (Chrome, Firefox, Safari)
- Use incognito/private mode
- Clear cookies and cache
- Try different time of day

### Wikimedia images won't download
- Make sure you click "Original file" link first
- Try "Download" button if available
- Right-click might be disabled - use browser's download option

### NARA catalog slow
- Searches can take 30+ seconds
- Be patient, don't refresh
- Download a few images, take break, come back

### Video files too large
- Choose HD instead of 4K if space is limited
- You only need 2-3 carabao clips total
- Don't need to download dozens of videos
