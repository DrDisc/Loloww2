# Text Overlay Specifications

**For:** Lolo's War - Philippine Scouts Video Series
**Created:** 2025-11-22
**Tool:** Create in Canva, Adobe Illustrator, or GIMP

---

## Design Guidelines

### Color Scheme
- **Primary Colors:** Red (#C41E3A) and Gold (#FFD700) - from Philippine Division insignia
- **Text Color:** White (#FFFFFF) or Cream (#FFF8DC) for readability
- **Background:** Semi-transparent black (#000000 at 60-70% opacity) for overlay

### Typography
- **Titles/Headers:** Stencil or bold sans-serif fonts
  - Recommended: Impact, Bebas Neue, Oswald Bold, Stencil
- **Body Text:** Clean sans-serif
  - Recommended: Arial, Helvetica, Roboto, Open Sans
- **Avoid:** Decorative, script, or hard-to-read fonts

### File Specifications
- **Format:** PNG with transparent background (for overlays)
- **Resolution:** 1920x1080 (16:9) or 1080x1920 (9:16 vertical)
- **Text Size:** Large enough to read on mobile (minimum 48pt for body text)
- **Animation Duration:** 2-3 seconds minimum on screen

## Animation Recommendations

### Simple Animations (Canva/PowerPoint)
- **Fade in/out:** 0.3-0.5 seconds
- **Slide in from bottom:** Lower third overlays
- **Scale up:** Center cards (start at 95%, scale to 100%)
- **Typewriter effect:** For quotes or important facts

### Professional Animations (After Effects)
- **Military-style transitions:** Quick wipes, hard cuts
- **Chart build-ins:** Organizational chart reveals line by line
- **Stat counters:** Numbers counting up (e.g., "11,972 soldiers")
- **Map overlays:** Labels appearing on tactical maps

---

## Accessibility Considerations

### Readability
- **Minimum contrast ratio:** 4.5:1 (text to background)
- **Text duration:** Long enough to read 2x (avg 250 words/min reading speed)
- **Font size:** Visible on small screens (test on mobile)

### Colors for Colorblind Viewers
- Red/Gold combination works well for most common colorblindness types
- Use strong contrast (white text on dark background) as primary
- Don't rely solely on color to convey information

## Production Workflow

### Step 1: Create Master Templates
1. Open Canva (free) or design software
2. Set canvas size: 1920x1080 or 1080x1920
3. Create color swatches: Red #C41E3A, Gold #FFD700
4. Test fonts: Download/install Bebas Neue, Oswald if needed

### Step 2: Create Text Overlays
1. Use specifications above for each overlay
2. Export as PNG with transparent background
3. Name files clearly: `script1_text_formed_1901.png`

### Step 3: Quality Check
- View at 100% size on monitor
- Check for typos
- Verify colors match Philippine Division insignia
- Test readability on dark and light backgrounds

### Step 4: Organize Files
Save to: `/media/graphics/text-overlays/`
Naming convention: `script[#]_[type]_[description].png`

Examples:
- `script1_text_formed_1901.png`
- `script2_org_chart_full.png`
- `script3_moh_recipients.png`

4. Replace text with specifications above
5. Download as PNG with transparent background

### Recommended Canva Elements
- **Shapes:** Rectangles with rounded corners for backgrounds
- **Lines:** For organizational charts and dividers
- **Effects:** Drop shadow on text for depth
- **Opacity:** Adjust background transparency
